Proximity

Udemy.com does a great job with using proximity in the design of the website. Though the image here isn't the best, you can see that there is good use of white space in the design. White space helps significantly in the readability of the page. Near the bottom of the page you can also see where it begins to start a new catagory for "What to learn next". The web site is very well set up like this by grouping together the various catagories of course available by subject matter. Courses can also be found grouped together according to what the user has signed up for. This all works together to aid in the usability and ease of navigation and readability of the website.
